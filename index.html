Overview

This app allows user to find the best rating restaurants that serve the dish that the user wish to have. Novelty of the Application Existing restaurant review forums, such as Yelp and Trip Advisor, only allow you to select the category of the food you wish to have. For example, you can only filter restaurants by categories like Chinese food, Spanish food, Indian food. However, sometimes we are looking for a restaurant that serves some specific dishes. When you try to look for a restaurant that serve dim sum, you will have to read through the menu or the reviews to find out whether this restaurant serves dim sum. Being able to filter the list by dishes is very helpful for this scenario. This application implements the function to allow user to search restaurant by the dish that they plan to have. Because of time limitation and simplification, this application only contains Chinese dishes to show the idea.

Intended User

The intended users of this application are restaurant finders who want to find a restaurant that serves the Chinese dish that they want to have.

Application Scenario

User A wants to have Dim Sum for diner. User A selects “Dim Sum” in the application, a list of restaurants that serves Dim Sum are shown. User A can pick the restaurant that best meets his interest from this application.

Interface Design

This application consists of three parts: dish selector, restaurant list and top rating graph. Dish selector is where the user can pick the dish that they plan to have. Restaurant list is a list of restaurants which serve the dish selected in the dish selector. In this list, the name, address and average rating are displayed. Rating graph part contains a graph that shows the rating of the top 10 restaurants that serve the selected dish. In this graph, the top 10 ratings of the restaurants that serve the selected dishes are shown.

Limitation

Because of time limitation, there are some limitations in this application: 1. This application only implements the simplest user interface to demonstrate the idea. There are lots of work can be done in the user interface to improve the application. For example, instead of using a table, a list that contains the photos of the restaurant will be better. 2. This application doesn’t show the reviews of the restaurant. Reviews can be added in future work so that it will better help the user to make dining decision. 3. Whether the restaurant serves the selected dish is based on the reviews text. More sophisticated methods can be used in future.
